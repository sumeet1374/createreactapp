# Create React App Shell Script  for Mac OS M1
# As original create-react-app was giving some error

Prerequisite

1) Install latest version of the node js
2) Install global json package -> npm install -g json


Creating a Project
Project will be created at home directory of the user. 
If you want to create in some other sub directory in the Home directory
When you run the shell script please provide project name as subdirectory/projectdirectory

Run following command to create react starter project

  source  cra.sh
